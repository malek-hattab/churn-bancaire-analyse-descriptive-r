# Analyse statistique descriptive du churn client bancaire

## Contexte
Ce projet vise à analyser le churn client dans un contexte bancaire à partir de données issues d’un même système d’information.

## Objectif
Réaliser une analyse statistique descriptive et comparative du churn client, selon une approche strictement non prédictive.

## Données
- CUSTOMERS : informations clients
- ACCOUNTS : comptes bancaires
- BALANCES : soldes financiers

## Méthodologie
- Fusion brute des tables
- Nettoyage fondé sur une logique métier bancaire
- Construction des variables âge et ancienneté
- Agrégation au niveau client
- Analyse descriptive et comparaisons statistiques (tests non paramétriques)

## Outils
- R / tidyverse
- Google Colab
- Quarto
- revealjs
